How do I manually remove Steam from my system if the uninstaller does not work?
As for a manual removal:
- delete the Steam directory
- delete the cache (if they were not in the Steam directory)
- delete shortcuts from the desktop
- delete shortcuts from start up menu
- from the registry HKEY_CURRENT_USER\Software\Microsoft\Windows\CurrentVersion\Run, delete the Steam value
- from the registry HKEY_LOCAL_MACHINE\Software\Microsoft\Windows\CurrentVersion\Uninstall, remove the Steam key

Does this software work with a DHCP connection?
Yes. The technology works with DHCP connections. Our electronic ticket system accounts for changes in your assigned IP address. If you IP address changes between sessions you will be asked to enter your password.


I try to open Steam and I get this message “Sorry- unable to connect to a Steam server. Please check your network connections and firewall settings.”
If you get this message, check your ports. Steam requires that your firewall have the following ports open:

Auto-updating:
TCP & UDP - 20
TCP & UDP - 21
Game play:
TCP - 2207
TCP - 5273
UDP - 27015

Users need to make sure the necessary ports are open in BOTH directions on their firewalls.
Ask them to send us their internet connection configuration, including the make/model of any NAT router
or connection sharing devise they use.
If your ports are open the Steam server may be down. Go to the News page for the latest info.


What is the minimum Internet connection speed required?
Term “broadband” refers to any Internet connection capable of sustained throughput of at least 256Kbps. We have designed it to deliver the optimum experience at 384Kbps. The technology will work at lower speeds but the experience will be slower and may require longer waiting periods before content is available. The only forms of phone line transmission omitted by this restriction is ISDN (128Kbps) and traditional dial-up modems (<56Kbps).


Can I play if my PC is behind a firewall?
Yes, but we may uses Internet ports that may be blocked by some firewalls.

You may have to reconfigure your networking equipment to open these ports. Home networks are made up of one or more computers behind a firewall, a router, a hub, or some combination of these.

A corporate firewall may cause problems running our system. We have designed security and password access systems that take this into account to try and improve accessibility. If you cannot access the games and your PC is behind a corporate firewall then you will need to discuss possible solutions with your corporate sys-admin.


Why does it seem to take extra time create a cache file on my PC?
Operating systems support NTFS and FAT32 file formats on the hard disk. If your operating system is Windows 2000 or Windows XP and the hard disk is configured in a FAT32 file format then you will see a delay in the creation of the cache file. Depending on network speeds this could take minutes to complete. Although it may not appear to be working the system is creating the cache file. This will only be seen the first time you create the cache file. This is not an issue when using NTFS file formats on all operating systems and FAT32 file formats on Windows 98 or Windows ME.

Where are the files stored on my PC?
As part of the start-up you will be asked to create a Steam directory. This directory can be placed anywhere on you local hard drive. The default location is C/:Steam/. All information is stored in this directory.

How do I change the screen settings on my computer?

  • Click the Start button.
  • Select Settings > Control Panel.
  • Double-click the Display icon to view the Display Properties control panel.
  • Click the Settings tab.

To change the number of colors your screen uses: In the Color Palette (Windows 95)/Colors (Windows 98) list, select the number of colors that you want your monitor to display.

To change the screen resolution: Under Desktop area (Windows 95)/Screen area (Windows 98), drag the slider until the correct resolution setting appears.

Note: Your monitor and display adapter determine the maximum number of colors that can appear on your screen, and whether you can change your screen resolution.

How much space will Steam take up on my hard drive if I accept the default cache size?
In general it depends on the application, however the default is selected for optimum application/game performance. In the case of Half-Life products, the default is 512MB.


Where is my account information stored?
Due to security issues, no account information is stored locally on the client, it is all stored on secure co-located servers in data distribution centers. Any type of transaction that requires user information is digitally encrypted before being sent over the internet thereby protecting this sensitive data. This personal information is used for Steam account/subscription maintenance only. At no time will this information be made available to anyone for any reason.

What is the purpose of the "Refresh" button?
This is only present in the beta version of the Steam UI. Use it only when directed to by Steam engineering as part of the beta process.


When I "right click" on one of the game icons I get a list of option. What is the purpose? Enable Cache Loading?
This menu contains a number of "advanced" features, one of which is "enable cache loading". This option instructs Steam to continue to use your internet connection when the application/game is not running and you are either using another application or your pc is idle. Enabling this option significantly improves your application/games performance.


When would I set the cache size smaller then recommended?
As mentioned above, reducing the size of the cache potentially reduces your application/games performance and should only be done when disk space is tight. Making the cache size smaller than 1/2 of it's recommended size is *highly* discouraged.


What is the system doing when it is down-loading the minimum footprint?
Even though the Steam components are installed on your computer and running, the application/game is not. There are some minimum number of files or components that the application/game needs in order to even display the splash screen. The files necessary to minimally start the game are referred to as the "minimum footprint" and are being acquired while this dialog is displayed. This is a short and one-time only process.


Why did the login fail after I entered my user name and special offer key?
The login screen requires you to enter your user name and password. During your account creation, you were asked to enter an email address, which you will use as your user name when logging in to the Steam application, a password, which you will use as your password when logging in to the Steam application, and a special offer key. The special offer key is used only during this registration process.


I prefer to not have desktop icons. Is there a way to prevent the Steam game icons from appearing on my desktop when I subscribe?
Yes. There is an option in the “Configuration” section of the Steam application, which disallows displaying Steam game icons on the desktop.


Are there any password restrictions?
Yes. Passwords must be at least six characters in length and must not contain more than thirty-two characters.


What is the difference between "Subscriptions" and "My Games"?
Subscriptions supplies users with a list of games available for subscription. “My Games” displays games in which you have subscribed.


Can I put my cache folder anywhere?
Yes. After you extract and open the Steam.exe, you have the option of creating a cache folder in the desired location.
