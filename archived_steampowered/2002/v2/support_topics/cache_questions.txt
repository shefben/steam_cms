									<p><font size="2" face="Arial,Helvetica,Geneva,Swiss,SunSans-Regular"><b>What is the purpose of Enable Cache Loading?<br>
											</b></font><font size="2" face="Arial,Helvetica,Geneva,Swiss,SunSans-Regular">This menu contains a number of "advanced" features, one of which is "enable cache loading". This option instructs Steam to continue to use your internet connection when the application/game is not running and you are either using another application or your pc is idle. Enabling this option significantly improves your application/games performance.<br>
											<br>
										</font></p>
									<p><font size="2" face="Arial,Helvetica,Geneva,Swiss,SunSans-Regular"><b>When would I set the cache size smaller then recommended?<br>
											</b></font><font size="2" face="Arial,Helvetica,Geneva,Swiss,SunSans-Regular">As mentioned above, reducing the size of the cache potentially reduces your application/games performance and should only be done when disk space is tight. Making the cache size smaller than 1/2 of it's recommended size is *highly* discouraged.<br>
											<br>
										</font></p>
									<p><font size="2" face="Arial,Helvetica,Geneva,Swiss,SunSans-Regular"><b>Why does it seem to take extra time create a cache file on my PC?<br>
											</b></font><font size="2" face="Arial,Helvetica,Geneva,Swiss,SunSans-Regular">Operating systems support NTFS and FAT32 file formats on the hard disk. If your operating system is Windows 2000 or Windows XP and the hard disk is configured in a FAT32 file format then you will see a delay in the creation of the cache file. Depending on network speeds this could take minutes to complete. Although it may not appear to be working the system is creating the cache file. This will only be seen the first time you create the cache file. This is not an issue when using NTFS file formats on all operating systems and FAT32 file formats on Windows 98 or Windows ME.<br>
