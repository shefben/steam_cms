UPDATE NEWS - ZEN OF SUDOKU
Latest update: March 01, 2007
Full Change History:
[-] March 01, 2007 (current release)
Changes/Additions
  • If you try to create a new profile with the same name as an existing one, the game will tell you so
  • Duplicate number hints are now more specific
  • Duplicate number and error hints can now be "explained" and are removed automatically with the Explain button
  • Made the x-wing, swordfish and jellyfish explanations clearer
  • Tweaked hint overlay fading speeds to make it a little faster
Bug fixes
  • Fixed bug that forced player to create new profile when starting the game if they had previously played v1.0
  • Fixed problem with jellyfish hints that could have caused a crash
  • Fixed bug where if you clicked the mouse button repeatedly while the "explain" overlay was fading away, sometimes it would fade back in
[+] February 27, 2007
Changes/Additions
  • Vista support (and support for reduced access accounts)
  • Improved hint system gives hints that always make sense (and it fills the hint in for you after explaining it)
  • Added ability to remove all pencil marks with the DEL key
  • Number chooser now fades in smoothly
  • Player-completed tiles no longer show a stone background so you can see your progress
  • Misc. graphic improvements to intro and credits
Bug fixes
  • Fixed bugs with custom games and made it easier to use (duplicates flash yellow, particles indicate completed puzzle, etc.)
[+] December 14, 2006
Changes/Additions
  • Released on Steam
