    Implementation Notes

    Keep interactive behavior server-side or add minimal vanilla JS as needed; the layout and look come from the theme.

    Use window-row to create two columns of windows. Nest box freely; combine with rounded for emphasis.

    Routing ideas (pseudocode):

GET /admin               -> dashboard overview
GET /admin/users         -> list users
POST /admin/users/:id    -> update user
GET /admin/content       -> list/edit pages
POST /admin/settings     -> apply configuration
